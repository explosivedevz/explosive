## 🔧 Troubleshooting & Common Issues

### ❌ Menu Won't Inject

**Cause:** Antivirus blocking the launcher or insufficient permissions.

**Fix:** Run as Administrator, add folder to AV exclusions, then retry.

***

### ❌ Game Crashes on Injection

**Cause:** Version mismatch between the menu and the current game build.

**Fix:** Check the Releases page for the latest compatible version and update immediately.

***

### ❌ Hotkey Not Responding

**Cause:** Hotkey conflict with another application running in the background.

**Fix:** Open `explosive_config.json` and change `menu_hotkey` to an unused key, then restart the launcher.

***

### ❌ ESP Not Displaying

**Cause:** ESP requires DirectX overlay compatibility. Some GPU driver versions interfere.

**Fix:** Update your GPU drivers to the latest version and ensure in-game overlay settings are not blocking third-party overlays.

***

### ❌ Android Companion Can't Connect

**Cause:** Firewall blocking the local network port used by the launcher.

**Fix:** Add an inbound firewall exception for port `7420` on Windows Defender or your third-party firewall.

***

### ❌ Config Changes Not Saving

**Cause:** File is set to read-only, or the launcher is overwriting it on close.

**Fix:** Right-click the config file → Properties → uncheck "Read-only", then re-apply your edits.
